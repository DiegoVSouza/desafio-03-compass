## 👨‍💻 Rodando na sua máquina

**1:** Clone o repositório.

```
git clone https://github.com/DiegoVSouza/catalog-front.git

```

**2:** Acesse o diretório do projeto.

```
cd catalog-front
```

**3:** Instale as depedências
```
  yarn

```
**4:** Inicie a aplicação.
```
  yarn start
   
```
